# Cohort9Website

Site is currently hosted at [nss-cohort-9.github.io](http://nss-cohort-9.github.io/)

##Setup
- git clone the repo
- Install bundler if you don't have it already with "gem install bundler"
- `bundle install`
- To run the server do "bundle exec jekyll serve"

##More Tips
